From Raw Heatmaps to ASD Classification

The complete pipeline: dataset, preprocessing, model architecture, and loss design.

01EDA 02CLAHE 03Bilateral Filter 04Binarisation / Morphology 05U-Net + SE Attention 06Residual CNN Classifier

The primary dataset is the public ASD Saliency Dataset (Zenodo DOI: 10.5281/zenodo.2647418): 300 stimulus images, 14 clinically diagnosed ASD children and 14 typically-developing controls, each viewing every image for a 3-second free-viewing trial. This yields 600 heatmaps (300 ASD + 300 TD) encoding fixation density as both scanpaths and overlay maps.

Dataset Summary
  • Stimuli images: 300
  • ASD participants: 14 (clinical diagnosis) · TD participants: 14
  • Trial duration: 3 seconds per image, free-viewing
  • Eye-tracking data: fixation maps + scanpaths (x, y, duration)
Custom GazeRecorder Dataset

To capture real-world variability, we collected our own webcam-based gaze data using GazeRecorder, recruiting volunteers from Reddit (r/autism, r/AutisticAdults). Each session was 30 seconds total: 10 stimulus images × 3 seconds each, producing gaze heatmap overlays on AMU campus landmark photographs.

Why this matters: GazeRecorder needs no specialised hardware — just a standard webcam — making the screening approach realistically deployable at scale.

Before committing to any architecture, we quantified low-level statistical differences between ASD and TD heatmaps across seven features: Entropy, Mean Intensity, Blob Spread, Contrast (RMS), Mean Blob Size, Number of Hotspots, and Hot Area %.

Key Observation

ASD heatmaps show higher entropy (4.81 vs 4.74) and larger mean blob size (1079.95px vs 1033.40px) than TD — gaze is more diffuse and spread over broader regions. TD heatmaps show more distinct hotspots (10.86 vs 10.45) — gaze clusters into more numerous, sharper focal points. Full statistical table is on the Results page.

Sample Code — Feature Extraction
from scipy.stats import entropy
import numpy as np

def heatmap_entropy(hmap):
    hist, _ = np.histogram(hmap.flatten(), bins=256, range=(0, 1), density=True)
    hist = hist / hist.sum()
    return entropy(hist + 1e-12)

def blob_spread(hmap, threshold=0.5):
    ys, xs = np.where(hmap > threshold)
    return np.std(xs) + np.std(ys)  # spatial spread of high-intensity region

CLAHE enhances local contrast by equalising the histogram within small tiles (e.g. 8×8), then clips and redistributes any bin exceeding a clip limit to prevent over-amplification of noise. Bilinear interpolation removes tile-boundary artefacts.

Formula

s = T(r) = (L−1) · CDF(r),   computed per-tile with clip_limit = β·(tile_size²/L)

Code Snippet
import cv2

clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
enhanced = clahe.apply(gray_heatmap)

Result: Consistently improved performance across all three pipelines — the single most reliable preprocessing step in this project.

The Bilateral Filter performs edge-preserving smoothing — it reduces noise while keeping fixation-region boundaries sharp, using both spatial proximity and intensity similarity.

Pipeline 1 (Full Complex — worst, 61.36%)

Bilateral → +Gaussian → +CLAHE → +Otsu Binarisation → +Adaptive Threshold → +Morphology → +Watershed

Pipeline 2 (Balanced — 82.31%)

Raw → +CLAHE → +Bilateral → +Min-Max Norm → +Binarisation

Pipeline 3 (Best — 83.08%)

Raw → +CLAHE → +Bilateral → +Min-Max Norm  no binarisation

Code Snippet
bilateral = cv2.bilateralFilter(clahe_img, d=9, sigmaColor=75, sigmaSpace=75)
norm = cv2.normalize(bilateral, None, 0, 1, cv2.NORM_MINMAX, dtype=cv2.CV_32F)

Conclusion: Skipping the final binarisation step (Pipeline 3) preserves continuous intensity gradients that carry diagnostic signal — binarising throws this away.

A U-Net encoder-decoder with Squeeze-and-Excitation (SE) attention blocks. Input: [B, 3, 128, 128] preprocessed stimulus image. Output: [B, 2, 128, 128] — predicted ASD gaze map + predicted TD gaze map.

Architecture Highlights
  • Encoder: 4 stages of Conv2d → SE Block → MaxPool2d (channels double, spatial size halves: 128→64→32→16→8)
  • Bottleneck: 8×8 spatial, 768 channels, Dropout2d(0.2)
  • Decoder: ConvTranspose2d upsampling + skip connections from matching encoder stage
  • Output: Sigmoid activation → valid probability map in [0,1]
SE Block Formula

z_c = GlobalAvgPool(F_c)  →  s = σ(W₂·δ(W₁·z))  →  F̃_c = s_c · F_c

Loss Function (Composite)

L_M1 = MSE + 0.1 × KL + 0.05 × (1 − Pearson r) 

import torch.nn as nn

class SEBlock(nn.Module):
    def __init__(self, channels, reduction=16):
        super().__init__()
        self.avg_pool = nn.AdaptiveAvgPool2d(1)
        self.fc = nn.Sequential(
            nn.Linear(channels, channels // reduction),
            nn.ReLU(inplace=True),
            nn.Linear(channels // reduction, channels),
            nn.Sigmoid()
        )

    def forward(self, x):
        b, c, _, _ = x.shape
        z = self.avg_pool(x).view(b, c)
        s = self.fc(z).view(b, c, 1, 1)
        return x * s

Takes a 3-channel overlay input [B, 3, 128, 128]: ch0 = actual patient heatmap, ch1 = Model 1's predicted ASD region, ch2 = ch0 × ch1 (element-wise product — the strongest classification signal, active only where real gaze coincides with the predicted discriminative zone). Output: [B, 2] logits (TD / ASD).

Architecture Highlights
  • 4 residual blocks with Channel Attention (avg + max pooling, more expressive than plain SE)
  • Global Average Pooling to [B, 256, 4, 4]
  • FC head with Dropout(0.4) → 2 output logits
  • CrossEntropyLoss with label_smoothing = 0.1
Channel Attention Formula

A = σ( MLP(AvgPool(F)) + MLP(MaxPool(F)) )  →  F̃ = A ⊗ F 

class OverlayClassifier(nn.Module):
    def __init__(self, num_classes=2):
        super().__init__()
        self.layer1 = ResidualBlock(3, 64, stride=2)
        self.layer2 = ResidualBlock(64, 128, stride=2)
        self.layer3 = ResidualBlock(128, 256, stride=2)
        self.layer4 = ResidualBlock(256, 256, stride=2)
        self.gap = nn.AdaptiveAvgPool2d((4, 4))
        self.fc = nn.Sequential(
            nn.Flatten(),
            nn.Linear(256 * 4 * 4, 512), nn.ReLU(), nn.Dropout(0.4),
            nn.Linear(512, num_classes)
        )

    def forward(self, x):
        x = self.layer4(self.layer3(self.layer2(self.layer1(x))))
        return self.fc(self.gap(x))

Metric Formulas
  • Accuracy = (TP+TN) / (TP+TN+FP+FN)
  • Precision = TP / (TP+FP)
  • Recall = TP / (TP+FN)
  • F1 = 2·(Precision·Recall) / (Precision+Recall)

Headline Result

Pipeline 3 (CLAHE + Bilateral) achieves 83.08% accuracy, 0.8136 F1-score, and 0.9057 precision — the best balance of sensitivity and false-positive control for a screening context.
